# 🎂 Age Calculator

A clean and elegant age calculator web app that tells you your exact age in years, months, and days — built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- Calculate your exact age in **years, months, and days**
- **Input validation** — catches empty fields, invalid dates, and future dates
- Displays your **next birthday** info
- Smooth animations and a soft pastel UI
- Fully **responsive** — works on mobile and desktop

## 🛠️ Technologies Used

- HTML5
- CSS3 (custom properties, animations, glassmorphism)
- Vanilla JavaScript (no libraries or frameworks)
- Google Fonts — Cormorant Garamond & Jost

## 📁 Project Structure

```
age-calculator/
├── index.html
├── style.css
└── app.js
```

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/codewithfiza/age-calculator.git
   ```
2. Open `index.html` in your browser — no setup needed!

## 📸 Preview

> Enter your date of birth → Click **Calculate My Age** → See your exact age instantly!
